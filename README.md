# Ticketing System

This project is a simple ticketing system implemented using Blazor WebAssembly. It allows users to create, view, and manage support tickets.

## Features

- Create new support tickets
- View a list of all tickets
- Update ticket status
- Responsive UI with Bootstrap styling

## Technologies Used

- Blazor WebAssembly
- C#
- Bootstrap

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Sanctana/CSIT221_Data-Structures-Project.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CSIT221_Data-Structures-Project
   ```
3. Open the project in your preferred IDE (e.g., Visual Studio, VS Code).

4. Run the application.

```bash
    dotnet watch run # Run the Blazor WebAssembly application with hot reload
    #or
    dotnet run # Run the Blazor WebAssembly application without hot reload
    #or
    dotnet publish -c Release -o ./publish # Publish the application for deployment

```
